What is Global Chat?

Global Chat allows users from different locations to communicate publicly. It's a space for sharing thoughts, asking questions, and engaging with the community.

build

How to Use:

  • Type your message in the input box and press Send.
  • Messages are visible to all users in real-time.
  • Use emojis, text, or reactions (if supported).
check_circle

Do's:

  • Be respectful and kind to others.
  • Use clear language to communicate.
  • Stay on topic if there's a designated theme.
cancel

Don'ts:

  • No hate speech, harassment, or bullying.
  • No spamming, flooding, or self-promotion.
  • Don't share personal/private information.
  • Avoid using all caps (seen as shouting).
lightbulb

Tips:

  • Use username to mention someone (if available).
  • Tap/Click a user's profile to view more or start a private chat.
  • Report inappropriate messages using the ⚠️ or ⋮ menu.
